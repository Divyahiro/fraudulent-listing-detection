# Fraudulent Listing Detection

Machine learning project to identify potentially fraudulent online transactions using classification techniques.

## ✅ Status
Completed

## Objective
Build a machine learning model to detect fraudulent transactions using transaction, user, and merchant-related features.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Train-Test Split
- Random Forest Model Training
- Model Evaluation
- Prediction Saving

## Repository Structure
- `data/`
- `notebooks/`
- `src/`
- `models/`

## Dataset
Fraud Detection E-Commerce dataset from Kaggle.

## Results
- Trained a Random Forest classifier.
- Evaluated the model using Accuracy, Classification Report, and Confusion Matrix.
- Saved predictions as `fraud_predictions.csv`.
